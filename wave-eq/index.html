2D WAVE EQUATION

A finite-element 2D wave equation solver, using a leapfrog integration scheme. Colours are assinged according to local value of displacement function. Implemented in javascript, graphics in webGL.

  • MOUSECLICK: generate wave.
  • UP/DOWN: adjust magnitude of wave.
  • LEFT/RIGHT: adjust width of wave.
  • SPACEBAR: toggle disturbance from moving cursor.
J. Lok, April 2016.
